## 🧠 About

This repository is a **personal engineering vault and continuous learning log** — not just notebooks, but production-quality implementations built with intent.

Every module here was built to answer one question: *"Do I actually understand this deeply enough to build it from scratch?"*

From implementing **KNN and Logistic Regression with pure NumPy math**, to training a **DQN agent that plays Flappy Bird**, to deploying **multi-agent AI systems with persistent memory** — this workbench documents the full journey of an AI/ML engineer.

```
Raw Data → Feature Engineering → ML Models → Deep Networks
     → Reinforcement Learning → GenAI Pipelines → Agentic Systems → Deployment
```

## 🏗️ What's Inside

### 🐍 Python & Data Fundamentals

> The foundation everything else is built on.

- Python basics, OOP patterns, and Pythonic idioms
- **NumPy** array operations, broadcasting, vectorized math
- **Pandas** DataFrames — cleaning, merging, groupby, time-series
- **Web Scraping** with BeautifulSoup
- Data visualization with **Matplotlib** and **Seaborn**

---

### 🤖 Machine Learning

> Every major algorithm — understood, not just imported.

| Category | Algorithms |
|---|---|
| **Supervised — Regression** | Linear, Lasso, Ridge Regression |
| **Supervised — Classification** | Logistic Regression, Naive Bayes, KNN, Decision Trees, SVM, Random Forest |
| **Ensembles** | AdaBoost, Gradient Boosting, XGBoost, Stacking, Voting, AutoML |
| **Unsupervised** | K-Means, DBSCAN, Hierarchical Clustering, Isolation Forest, PCA |
| **From Scratch** ⭐ | KNN, Linear Regression, Logistic Regression — pure NumPy math, no ML libraries |

- Bias/Variance Tradeoff · Precision, Recall, F1 · Cross-validation
- Feature Engineering & Imputation · Hyperparameter Tuning

## ⚡ Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/Sumit07125/AI_ML_Workbench.git
cd AI_ML_Workbench

# 2. Create a virtual environment
python -m venv venv
source venv/bin/activate      # Linux/Mac
# venv\Scripts\activate       # Windows

# 3. Install core dependencies
pip install numpy pandas matplotlib seaborn scikit-learn torch
pip install langchain chromadb sentence-transformers openai groq
pip install fastapi flask streamlit uvicorn agno

# 4. For the Flappy Bird DQN agent
pip install flappy-bird-gymnasium

# 5. Set up API keys (for GenAI modules)
export OPENAI_API_KEY="your-key-here"
export GROQ_API_KEY="your-key-here"
```

> 💡 **Tip:** Each module folder contains its own `requirements.txt` and a brief `README` explaining usage.
